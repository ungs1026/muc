기본 사용법 속도 & 딜레이 반복 옵션 커서 & 백스페이스 컨트롤 메서드

Typed.js 기능 정리 및 예제 프리뷰

기본 사용법 (Basic Usage)

가장 기본적인 형태로, `strings` 배열에 담긴 문자열을 순서대로 타이핑합니다.

const options = {
    strings: [
        'First sentence.', 
        'Second sentence.',
        '안녕하세요, Typed.js 입니다.'
    ],
    typeSpeed: 50,
};

const typed = new Typed('#typed-basic', options);

속도 & 딜레이 옵션 (Speed & Delay)

`typeSpeed`, `backSpeed`, `startDelay`, `backDelay` 옵션을 통해 타이핑, 삭제, 시작, 재시작 시의 속도와 지연 시간을 제어할 수 있습니다.

new Typed('#typed-speed', {
    strings: ['Typing slowly...', 'Deleting slowly...', 'And starting again after a delay.'],
    typeSpeed: 100,      // 타이핑 속도 (느리게)
    backSpeed: 100,      // 삭제 속도 (느리게)
    startDelay: 1000,    // 시작 전 딜레이
    backDelay: 2000,     // 삭제 후 딜레이
    loop: true
});

반복 옵션 (Loop)

`loop: true`로 무한 반복을 설정하거나, `loopCount`를 지정하여 특정 횟수만큼만 반복시킬 수 있습니다. (아래 예제는 3회 반복)

new Typed('#typed-loop', {
    strings: ['Looping.', 'Again.', 'One more time.'],
    typeSpeed: 50,
    backSpeed: 50,
    loop: true,
    loopCount: 3 // 3회만 반복
});

커서 & 스마트 백스페이스 (Cursor & Smart Backspace)

`cursorChar`로 커서 모양을 바꾸거나, `smartBackspace: false`로 설정하여 공통 부분까지 모두 지우도록 만들 수 있습니다.

Custom Cursor:
Smart Backspace Off: 
// 1. Custom Cursor
new Typed('#typed-cursor', {
    strings: ['Cursor is now an underscore.'],
    typeSpeed: 50,
    cursorChar: '_', // 커서 문자 변경
    loop: true
});

// 2. Smart Backspace Off
new Typed('#typed-backspace', {
    strings: ['I am a Vue developer.', 'I am a React developer.'],
    typeSpeed: 50,
    smartBackspace: false, // 스마트 백스페이스 비활성화
    loop: true
});

컨트롤 메서드 (Control Methods)

생성된 Typed 인스턴스의 메서드를 호출하여 애니메이션을 직접 제어할 수 있습니다.

const controlledInstance = new Typed('#typed-control', { /* ... */ });

// Play Button
function playTyped() {
    controlledInstance.start();
}
// Pause Button
function pauseTyped() {
    controlledInstance.stop();
}
// Reset Button
function resetTyped() {
    controlledInstance.reset();
}
